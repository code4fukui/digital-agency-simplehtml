# digital-agency-simplehtml

デジタル庁のウェブサイトを Textブラウザや音声ブラウザでも利用できるよう、SimpleHTML に変換したプロジェクトです。

## デモ
このリポジトリの GitHub Pages で確認できます: https://code4fukui.github.io/digital-agency-simplehtml/

## 機能
- デジタル庁のホームページ、組織情報ページを SimpleHTML に変換
- その他のページも順次変換中

## 必要環境
特にありません。

## 使い方
このリポジトリをクローンして、ご自身で変換作業にご協力ください。
変換ページの追加や修正などがあれば、プルリクエストをお送りください。
